The Trim Galore logos are free to use. Each comes in a light variant (for placement on light surfaces) and a dark variant (for placement on dark surfaces). Both **SVG** (vector — infinite resolution) and **PNG** (raster — easier to paste into slides, README files, and chat tools that don't render SVG) are provided.

The paper and ink colours are pure `#FFFFFF` and `#0d1117` so the logos sit cleanly on default light/dark backgrounds (matching GitHub's surfaces too). The lime accent `#C2EE49` is preserved across both variants.

## Hero

The 16:9 wordmark with the diagonal lime band — used as the homepage hero and the GitHub README banner. PNG renders at 1920×1080.
